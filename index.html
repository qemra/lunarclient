LunarClient — Static Landing

This page confirms Netlify can serve static assets from this repository.

If your project requires the Express proxy (index.js), Netlify static hosting will not run that server — use Render/Vercel/Railway or convert the backend to serverless functions.

  • To serve this file from Netlify, place it in the repo root or in public/ and set Netlify's publish directory accordingly.
  • If you use a SPA with client-side routes, also add public/_redirects containing /* /index.html 200.
  • If you deploy a Node server instead, update index.js to use process.env.PORT and host on a platform that supports Node processes.
